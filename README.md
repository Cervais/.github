# .github

Community health files and organization-wide defaults for [Cervais](https://cervais.com) repositories on GitHub.

GitHub automatically applies the files in this repository to every public repository in the [cervais](https://github.com/cervais) organization that doesn't define its own copy. That includes:

| File | Purpose |
|---|---|
| [`profile/README.md`](profile/README.md) | The organization's public landing page, shown on [github.com/cervais](https://github.com/cervais). |
| [`SECURITY.md`](SECURITY.md) | How to report a security vulnerability. |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Expected behavior for contributors and maintainers. |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to propose changes to Cervais repositories. |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Default issue templates (bug report, feature request). |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | Default pull request template. |

Individual repositories can override any of these by adding their own copy of the same file.
